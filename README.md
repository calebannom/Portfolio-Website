


# Portfolio-Website
A dark, hacker-themed personal portfolio website built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, just clean code.



This is my personal portfolio site, designed to showcase my skills, projects, and background as a junior software engineer. The aesthetic is inspired by terminal interfaces and hacker culture — dark backgrounds, green accent colors, monospace fonts, and a matrix rain animation in the hero section.
The site is fully static (HTML + CSS + JS), making it easy to host for free on platforms like Netlify or Vercel with zero configuration.

Features

Matrix rain hero — animated falling characters in the background
Typing effect — cycles through different role titles automatically
Scroll reveal animations — sections fade in as you scroll down
Skill bar animations — progress bars animate when they enter the viewport
3D card tilt — project cards tilt on mouse movement
Active nav highlighting — nav links highlight based on scroll position
Hamburger menu — responsive mobile navigation
Contact form — client-side form with submit feedback
Fully responsive — works on desktop, tablet, and mobile
